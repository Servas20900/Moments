React 18 + Vite + TypeScript single page app for Moments, a luxury vehicle with driver service. The visual system uses dark luxury styling, CSS variables, and Cloudinary images.

## Scripts

- `npm run dev` start dev server
- `npm run build` production build
- `npm run lint` lint with ESLint

## Structure

- src/pages Home, Paquetes, Calendario, Evento especial, Contacto
- src/components Button, Card, Modal, CalendarDay, VehicleCard, PackageCard, SafeImage
- src/data/content.ts sample data and Cloudinary public ids

## Notes

- Cloudinary assets use demo public ids; replace with brand assets when available.
- Typography uses Playfair Display and Manrope via Google Fonts.
- Colors, radii, shadows, and transitions are defined as CSS variables in src/index.css.
